Data Visualization Project

"Analisis Dampak Serangan Terorisme di Kawasan Asia (1982-2020)"

Ringkasan Eksekutif: Dalam proyek ini, saya dan tim melakukan eksplorasi data mendalam terhadap fenomena terorisme di Asia menggunakan Global Terrorism Database. Fokus utama kami adalah mentransformasi data mentah yang kompleks menjadi narasi visual yang mudah dipahami untuk mengidentifikasi pola serangan selama 38 tahun terakhir.

1. Tujuan Analisis

Tujuan utama dari visualisasi ini adalah untuk memetakan distribusi geografis serangan, menganalisis fluktuasi jumlah korban jiwa, dan mengidentifikasi jenis senjata yang paling sering digunakan, guna memberikan gambaran historis yang jelas mengenai stabilitas keamanan di Asia.

2. Metodologi & Tools

Saya menerapkan alur kerja pengolahan data sebagai berikut:

  • Sumber Data: Global Terrorism Database (GTD) periode 1982-2020.
  • Tools Visualisasi: Menggunakan Tableau untuk dashboard interaktif dan Python (Matplotlib/Seaborn) untuk analisis statistik awal.
  • Proses: Data Cleaning (pembersihan nilai kosong), Data Filtering (fokus wilayah Asia), dan Pembuatan Visualisasi (Map, Trendline, Bar Chart).

3. Temuan Utama Melalui Visualisasi

Melalui dashboard yang saya bangun, ditemukan beberapa poin krusial:

  • Tren Serangan: Terjadi peningkatan signifikan serangan pada periode tertentu, dengan titik puncak yang dipengaruhi oleh ketegangan geopolitik regional.
  • Wilayah Terdampak: Visualisasi peta (Heatmap) menunjukkan konsentrasi serangan tertinggi berada di wilayah Asia Selatan dan sebagian Asia Tenggara.
  • Metode Serangan: Penggunaan bahan peledak dan senjata api tetap menjadi metode dominan yang menyebabkan dampak kerusakan terbesar.

4. Kesimpulan Visualisasi

Proyek ini membuktikan bahwa visualisasi data adalah alat yang sangat kuat dalam memahami isu global yang sensitif. Dengan grafik yang tepat, pola tersembunyi dari data terorisme dapat diungkap untuk membantu pemangku kebijakan dalam melakukan penilaian risiko di masa depan.
